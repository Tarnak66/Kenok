# Kenok - Useful AI 🤖
Kenok е интелигентен асистент, изграден с Python, Streamlit и Groq (Llama 3.3). 
Той може да търси информация в Google в реално време и да води разговори с памет.

# [![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](ТУК_ПОСТАВИ_ТВОЯ_ДИРЕКТЕН_ЛИНК)
Кликни върху бутона gore, за да използваш Kenok директно в браузъра!
